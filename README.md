About GRACE Work Vault
The GRACE Work Vault is a student-created webpage made to compile all current assignments into one easy-to-access location. Each assignment is automatically sorted into categories by the first alphanumeric character (0-9, A-Z) in the assignment's name. For example, an assignment called "Spanish-Duque El Heroe" would be placed in the "S" category, as the assignment name starts with an "S."
If no assignment begins with a specific alphanumeric character, and that folder is opened on the vault, the text "No Files" will appear under said folder.

Version Format
Version numbers follow the format w.x.y.z
Micro Update: z represents a small, behind-the-scenes update with minimal visible changes.
Update: y indicates a larger update that improves functionality but doesn’t significantly alter the site.
Sub Version: x denotes a major update with substantial changes or new features.
Full Version: w signifies an entirely new version of the site.
